Gathering-RU
======

Отображение уведомлений в чате и предупреждений на экране о найденных ресурсах.
![Preview](https://github.com/PinguinRei/Gathering-RU/blob/master/tip.PNG)

/8 команда | Описание эффекта
--- | ---
сбор | Вкл / Выкл Модуль (по умолчанию) [Вкл]
сбор предупреждение | Предупреждающее сообщение на экране [sendToAlert]
сбор уведомление | Сопровождающая подсказка в уведомлении [sendToNotice]
сбор статус | Запросить статус модуля
сбор Руда | Вкл / Выкл подсказки о найденной [Руде] (по умолчанию) [Выкл]
сбор Растения | Вкл / Выкл подсказки о найденных [Растениях] (по умолчанию) [Выкл]
сбор Энергия | Вкл / Выкл подсказки о найденной [Энергии] (по умолчанию) [Выкл]
